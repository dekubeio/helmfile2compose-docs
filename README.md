# helmfile2compose-docs

User documentation for the [helmfile2compose](https://github.com/dekubeio/helmfile2compose) distribution.

Published at **[helmfile2compose.dekube.io](https://helmfile2compose.dekube.io/docs/)**.

- `index.html` — static landing page
- `docs/` — MkDocs site (getting started, configuration, operations, known workarounds)

Part of the [dekube](https://dekube.io) project.

## Local preview

```bash
pip install mkdocs-material
mkdocs serve
```
